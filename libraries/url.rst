=============
URL 编码类
=============


+-------------+------------+
|属性         |值          |
+=============+============+
|命名空间     |fize\\crypt |
+-------------+------------+
|类名         |Url         |
+-------------+------------+


:方法:


+---------------+------------------------------------------+
|方法名         |说明                                      |
+===============+==========================================+
|`parse()`_     |解析 URL，返回其组成部分                  |
+---------------+------------------------------------------+
|`rawDecode()`_ |对已编码的 URL 字符串进行解码             |
+---------------+------------------------------------------+
|`rawEncode()`_ |按照 RFC 3986 对 URL 进行编码             |
+---------------+------------------------------------------+
|`decode()`_    |解码已编码的 URL 字符串                   |
+---------------+------------------------------------------+
|`encode()`_    |编码 URL 字符串                           |
+---------------+------------------------------------------+


方法
======
parse()
-------
解析 URL，返回其组成部分

.. code-block:: php

  public static function parse (
      string $url,
      int $component = -1
  ) : array|string|int


:参数:
  +----------+--------------------------------------------------------+
  |名称      |说明                                                    |
  +==========+========================================================+
  |url       |要解析的 URL。无效字符将使用 _ 来替换。                 |
  +----------+--------------------------------------------------------+
  |component |指定返回的部分                                          |
  +----------+--------------------------------------------------------+
